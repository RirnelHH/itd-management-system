# 教学计划增强优化设计（2026-04-17）

**对应变更**: `teaching-plan-enhancement-20260417`

## 设计目标
在教学计划主链路已经修通的基础上，继续面向真实使用场景优化编辑、反馈和界面一致性，使教学计划模块更接近最终可用状态。

## 设计重点
### 1. 表格编辑体验增强
- 优化教学计划表格中的新增、编辑、删除入口
- 提升学期分区与表格可读性
- 尽量减少“需要脑补结构”的操作成本

### 2. 导入导出反馈增强
- 成功提示增加结果摘要
- 失败提示更明确指向问题原因
- 对常见模板/数据异常提供更可读反馈

### 3. UI 一致性与可读性
- 统一教学计划及相关页面的按钮、表格、表单视觉体验
- 优化深色主题下的可读性与操作层级

### 4. 收尾治理
- 整理本轮 OpenSpec
- 处理本次开发遗留的生成文件/草稿文件
- 保持仓库提交干净
