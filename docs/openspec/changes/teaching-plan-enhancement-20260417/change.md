# 教学计划增强优化（第二轮）

**变更日期**: 2026-04-17
**变更类型**: 功能增强 / 体验收口
**影响范围**: 教学计划编辑器、导入导出反馈、UI 可读性、验收收尾
**状态**: 规划中

## 变更背景
上一轮已完成教学计划模板返修主链路：正式模板资源化、教学计划详情页表格化、任课老师字段删除、周课时从课程带出、导入反馈修复，并已推送主改动。当前进入第二轮增强，目标是围绕实际使用体验继续收口，提升教学计划编辑器的直观性、导入导出反馈质量和整体 UI 一致性。

## 变更范围
1. 增强教学计划表格编辑体验（更顺手的新增/编辑/删除、学期维度更清晰）。
2. 继续补强导入导出反馈与异常提示。
3. 优化教学计划页面及相关教研教学页面的 UI 可读性与交互一致性。
4. 整理并收口本轮 OpenSpec 与相关残留文件。

## 验收标准
- 教学计划表格编辑体验进一步提升，实际操作更直观。
- 导入成功/失败反馈更完整、可读。
- 教研教学页面 UI 一致性更好。
- 仓库残留文件与 OpenSpec 变更整理完成。
