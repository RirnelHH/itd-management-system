# 教研教学模块第一批

**变更日期**: 2026-04-11
**变更类型**: 功能新增
**影响范围**: 教研教学模块后端能力与数据结构
**状态**: 规划中

---

## 变更背景

为支撑教研教学模块的后续建设，需要先落地第一批基础后端能力，覆盖专业、课程、年级、实施性教学计划及计划明细行的核心数据结构与管理边界。

本批仅定义并交付后端接口与数据结构规则，不包含前端页面、Excel 导入导出、打印、报表或批量模板能力。

---

## 变更范围

### 1. 专业与课程

- 新增 `Major` 的基础管理规则
- 新增 `Course` 的基础管理规则
- 明确专业与课程之间的关联边界
- 支持按状态、归属和编码进行基础查询

### 2. 年级与实施性教学计划

- 新增 `Grade` 的基础管理规则
- 新增 `TeachingPlan` 的基础管理规则
- 新增 `TeachingPlanRow` 的行级结构规则
- 明确年级与实施性教学计划之间的一对多管理关系

---

## 本批不包含

- 前端页面、交互、路由与权限展示
- Excel 导入、导出、模板下载、批量填报
- 打印、报表、统计分析与仪表盘
- 审批流、工作流、消息通知
- 复杂排课、课表编排、课时自动平衡

- 明确本批仅涉及后端与数据结构，不延伸至前端与 Excel
